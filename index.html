VS Code nima?

VS Code (Visual Studio Code) — Microsoft tomonidan yaratilgan bepul kod muharriri. Dunyodagi eng mashhur dasturlash dasturi!

O'rnatish qadamlari:

  • 1. code.visualstudio.com saytiga kiring
  • 2. "Download for Windows" tugmasini bosing
  • 3. Yuklab olingan .exe faylni ishga tushiring
  • 4. "Next" tugmalarini bosib o'rnating

Kerakli Extensions (qo'shimchalar):

  • 🔴 Live Server — saytni brauzerda ko'rsatadi
  • 🔵 Prettier — kodni tartibga keltiradi
  • 🟡 Auto Rename Tag — HTML teglarni avtomatik o'zgartiradi

Extension o'rnatish:

VS Code da Ctrl+Shift+X bosing → qidiruv qatoriga Extension nomini yozing → "Install" bosing ✅

HTML nima?

HTML (HyperText Markup Language) — veb-sahifalarning asosiy tili. Sahifaning tuzilmasini belgilaydi.

Asosiy tuzilma:

index.html
<!-- Har bir HTML fayl shunday boshlanadi --> <!DOCTYPE html> <html lang="uz"> <head> <!-- Ko'rinmaydigan sozlamalar --> <meta charset="UTF-8"> <title>Mening Saytim</title> </head> <body> <!-- Ko'rinadigan qism --> <h1>Salom Dunyo! 👋</h1> <p>Bu mening birinchi sahifam</p> </body> </html>

Sinab ko'ring:

  • VS Code oching
  • Yangi fayl yarating: Ctrl+N
  • Yuqoridagi kodni yozing
  • index.html deb saqlang: Ctrl+S
  • Live Server bilan oching: o'ng klik → "Open with Live Server"

CSS — Cascading Style Sheets

CSS yordamida HTML elementlarning ko'rinishini o'zgartiramiz: rang, o'lcham, joylashuv va boshqalar.

CSS ni HTML ga ulash:

index.html
<head> <!-- CSS faylni ulash --> <link rel="stylesheet" href="style.css"> </head>
style.css
/* Barcha elementlarga */ * { margin: 0; padding: 0; } /* body ga stil */ body { background: #111; color: #fff; font-family: Arial, sans-serif; } /* h1 ga stil */ h1 { color: #38bdf8; font-size: 2rem; text-align: center; }

JavaScript nima?
